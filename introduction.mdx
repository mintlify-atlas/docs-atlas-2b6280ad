# iMessage & Mail

iMessage & Mail is a [Claude Code](https://docs.anthropic.com/en/docs/claude-code) skill that gives Claude direct access to your macOS Messages and Apple Mail data. It reads from local SQLite databases and sends messages via AppleScript — everything runs on your Mac, nothing leaves your machine.

## How it works

The skill is an orchestrator (`SKILL.md`) that classifies your intent and routes to one of six specialized agent markdown files. Each agent executes SQLite queries against local macOS databases and uses AppleScript to control Messages.app and Mail.app for sending.

```
imessage-mail/
├── SKILL.md                    # Orchestrator — intent classification and routing
└── agents/
    ├── recent-activity.md      # Recent contacts and email overview
    ├── conversation-reader.md  # Full message history for a contact
    ├── message-search.md       # Cross-source keyword search
    ├── attachment-finder.md    # Find files and media from messages/emails
    ├── mail-reader.md          # Apple Mail database queries
    └── send-message.md         # Send iMessages and emails via AppleScript
```

## Privacy

This skill runs entirely locally on your Mac. No message data, contacts, or email content is sent to any external service beyond what Claude Code normally processes. All database queries are read-only except for the send functionality, which uses AppleScript to send via your local apps.

<Note>
  This skill requires **Full Disk Access** granted to your terminal application in macOS System Settings. See [Permissions](/setup/permissions) for setup instructions.
</Note>
